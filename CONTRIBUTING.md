# Contribuir a mindboard

¡Gracias por querer aportar! El proyecto es **un solo archivo** (`index.html`) con
HTML + CSS + JavaScript vanilla. No hay build ni dependencias.

## Cómo trabajar

1. Hacé un **fork** del repo.
2. Editá `index.html`.
3. Probalo abriéndolo en el navegador (doble clic). Verificá que la consola quede limpia.
4. Abrí un **Pull Request** contra `main` describiendo el cambio.

## Reglas

- `main` está **protegida**: no se puede pushear directo. Todo entra por PR y necesita
  al menos **una aprobación** del/los mantenedor/es antes de mergear.
- Mantené el estilo del código existente (vanilla JS, sin frameworks, sin CDN).
- Los datos del usuario viven en `localStorage`; **no** incrustes datos dentro del HTML.
- Cambios acotados y enfocados: un PR, un tema.

## Ideas para contribuir

- Mejoras de accesibilidad y mobile.
- Internacionalización (hoy la UI está en español).
- Nuevas exportaciones (Markdown, CSV).
- Tests.

¿Dudas o ideas grandes? Abrí un **Issue** primero para charlarlo.
